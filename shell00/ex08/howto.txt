В файле с именем clean поместите командную строку, 
которая будет искать все файлы - в текущем каталоге, 
а также в его подкаталогах - заканчивая символом ~ и / 
или всеми файлами, которые начинаются или заканчиваются символом #. 
Командная строка покажет и удалит все найденные файлы. 
Допускается только одна команда: нельзя использовать ; или && 
или другие махинации.

man find


Проверка:
cd ex08
ls
touch ff~ dd~ #ss# #ee#
ls
bash clean
ls


// Test: 
// cat clean 
// 
// Output: 
// #!/bin/sh find . -type f -name "##" -print -delete -o -type f -name "~" -print -delete

For more info:
man find
