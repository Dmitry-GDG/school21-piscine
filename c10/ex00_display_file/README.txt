• Создайте программу с именем «ft_display_file», которая отображает 
в стандартном выводе только содержимое файла, заданного в качестве аргумента.
• В каталоге отправки должен быть «Makefile» со следующими правилами: 
«all», «clean», «fclean». Двоичный файл будет называться «ft_display_file».
• Функция "malloc" запрещена. Вы можете сделать это упражнение, 
только объявив массив фиксированного размера.
• Все файлы, указанные в качестве аргументов, будут действительными.
• Если аргумент не указан, программа должна отображать:
	:	File name missing.
• Если аргументов более одного, программа должна отображать:
	:	Too many arguments.
• Если файл не может быть прочитан, программа должна отображать:
	:	Cannot read file.
